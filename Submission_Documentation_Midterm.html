1. Project Evolution: From HTML to CSS

This project represents the transition from a basic structural webpage (Module 3) to a fully-designed, premium digital experience. By applying modern CSS principles, the website was transformed from a plain black-on-white document into an "Obsidian Noir" portfolio.

MODULE 3 (Basic HTML)

Simple tags <h1>, <ul>, and <p>. Standard browser font (Times New Roman). Vertical flow only. No colors or depth.

MIDTERM (Enhanced CSS)

Custom Flexbox layouts. Google Fonts integration (Outfit). Glassmorphism effects. Dark theme UI. Interactive form logic.

2. Key Visual Enhancements

A. Premium Typography & Branding

Replaced default browser fonts with the 'Outfit' Google Font family to create a modern, executive feel. Added a gold accent color (#FFD700) to highlight key skills and status.

B. Responsive Skills Grid

Utilized CSS Grid to create a symmetric, three-column layout for technical expertise. This ensures the information is scannable and visually balanced compared to a simple bulleted list.

.skills-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 40px; }

C. Experience Timeline (Positioning & Borders)

Applied relative positioning and custom border-left properties to create a professional timeline effect for the work experiences.

3. Interaction & Layout Logic

Navigation Bar

Implemented a sticky navigation bar with glassmorphism using backdrop-filter: blur(). This allows users to access sections at any time while maintaining a premium feel.

Mobile Responsiveness

Used Media Queries to ensure the website stacks beautifully on mobile devices, converting the multi-column sections into vertical single-column layouts for touch devices.

Contact Form Styling

Designed a minimal, clean contact form using CSS transitions and focused state effects to enhance user experience.
